# TODO 收口与技术决策

最后更新：2026-07-11

本文记录 `todo.md` 中需要评估而非立即堆叠功能的事项。决策适用于暑期 50 人以内测试阶段，达到触发条件后重新评估。

## 主题与可访问性

- 当前维持单一深空主题，不增加完整亮暗模式或第二套高对比主题。
- 全站交互控件使用统一 `focus-visible` 外框，弹窗通过 `useDialogA11y` 处理 Esc、滚动锁定、初始焦点和焦点回收。
- 系统 `prefers-reduced-motion` 生效时关闭 CSS 持续动画；Canvas 星空只绘制静态帧。
- 新代码必须使用语义令牌。`npm run audit:ui-tokens` 固定现有历史债务上限，禁止新增硬编码颜色；旧页面按实际修改范围逐步迁移，避免一次性改写 50 余个页面造成视觉回归。

重新评估主题切换的触发条件：收到明确的低视力可读性反馈，或学校提出正式品牌主题规范。

## 联系方式存储

当前不实施应用层字段加密，原因如下：

- 联系方式为选填，前台默认仅本人可见；公开校友查询不返回联系方式。
- 管理员导出需要受控明文，应用层加密会引入密钥轮换、备份恢复和灾难恢复的新风险。
- 当前安全重点是最小字段返回、管理员鉴权、导出审计、文件使用提示和服务器磁盘/备份权限。

重新评估加密的触发条件：正式接收大量手机号、地方分会出现分级管理员，或组织取得独立密钥管理与恢复能力。

## 图片与对象存储

- 上传继续使用 Sharp 重编码和站内 `/uploads/` 目录。
- 列表与详情通过 Next Image 的 `sizes` 生成不同响应尺寸，不在磁盘重复维护缩略图。
- `npm run audit:images` 输出图片体积、尺寸和超过 2MB 的文件。
- 生产使用 Nginx 缓存 `/_next/static`、`/leaflet` 和 `/uploads`。

当前不迁移 OSS/COS。触发条件：上传目录超过 5GB、月流量超过服务器带宽预算、需要多实例部署，或备份恢复时间不可接受。

## 缓存策略

| Key 前缀 | TTL | 失效策略 |
| --- | --- | --- |
| `published:news:list:*` | 60 秒 | 新闻创建、更新、删除后按前缀删除 |
| `published:news:detail:*` | 60 秒 | 新闻创建、更新、删除后按前缀删除 |
| `published:events:list:*` | 60 秒 | 活动变更、报名或取消后按前缀删除 |
| `published:events:detail:*` | 60 秒 | 活动变更、报名或取消后按前缀删除 |
| `api:alumni:map` | 300 秒 | 短 TTL；名册批量变更后最多等待 5 分钟 |
| `admin:stats` | 60 秒 | 后台工作台短 TTL |

Redis 未配置或异常时直接查询数据库，缓存不是可用性依赖。

## 依赖与发布节奏

- Next.js、Prisma、React、Sharp、bcryptjs 每季度安排一次升级窗口；高危公告立即处理。
- 每周 CI 及发布前执行 `npm run audit:prod`。Next 内置 PostCSS 的中等公告等待官方兼容补丁，不通过 overrides 强行替换 Next 内部构建依赖。
- 轻量动效继续使用 CSS、Canvas 和原生 API，不引入 Framer Motion 等大型默认依赖。
- PR 必须通过类型、lint、契约测试、生产依赖审计、构建和 smoke。GitHub 分支保护中的“Require status checks”属于仓库设置，需要仓库管理员保持启用。

## 星空彩蛋

独立路由 `/starfield` 已完成完整互动演出，保持不加入常规导航且不允许搜索引擎索引。首页站点标识连续点击三次可触发隐藏入口。演出由同一组星点连续重组成工程师、协作建造场景、通用火箭、点火尾焰和升空画面，再衔接无文字会徽星点编队、轮廓连线、纪念文案揭示和重播。人物与火箭层不使用实体填充或连续 SVG 路径，使用现有 Canvas 星空、SVG 星点与 CSS 动画，不引入大型动画依赖。

会徽按外圈、内圈、轨道和中心图案分层汇聚，完整演出约 16 秒。时间节点集中定义在 `StarfieldExperience`，人物、建造、成型、点火、发射与会徽演出共享同一状态机。发射次数只写入浏览器本地存储，不进入数据库；首次观看完整播放，完成一次后在重播和后续访问中提供跳过按钮。

`RocketWorkshop` 固定维护 340 颗星，为各阶段生成目标坐标并通过最近邻匹配减少重组时的交叉飞行。拿到正式火箭图片后，应离线提取轮廓和内部特征点，替换通用火箭的目标点阵；人物、星点明暗节奏、点火、升空、会徽和整场时间线无需重写。工作坊退场后停止自身星点呼吸动画，避免与会徽编队同时占用移动端动画资源。

页面在 375×667、390×844 和桌面视口保持单屏无溢出；系统启用 `prefers-reduced-motion` 时跳过升空动画并直接显示文案。后续可在不改变状态机的前提下增加校友会品牌火箭素材或由用户主动开启的音效。
