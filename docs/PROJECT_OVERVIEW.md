# 燕中校友数字母港

- 正式校名：燕川中学
- 域名：yanchuaner.cn
- 服务器：华为云 ECS (Ubuntu)

## 定位

校友会展示、活动发布、活动报名、校友查询、后台运营的公益数字平台。非官方、无盈利。

## 核心功能

| 功能 | 说明 |
|------|------|
| 普通口令入口 | 校友/访问者输入内测口令后进入网站 |
| 管理员后台 | 管理员账号 + HMAC-SHA256 签名 token 登录 |
| 新闻管理 | 后台编辑、发布新闻；公开可访问 |
| 活动管理 | 后台编辑、发布活动；公开可访问，支持在线报名 |
| 活动报名名单导出 | 管理员导出 CSV（含 UTF-8 BOM） |
| 校友名单管理 | 后台 CRUD + CSV 导入/导出 |
| 校友查询 | 公开搜索接口，按姓名/届别/标签搜索 |
| 校友地图 | 从 tags 解析城市信息做分布展示 |
| 图片上传 | 管理员上传活动/新闻封面图片 |
| 联系我们 | 公开联系表单（邮箱引导）|

## 技术栈

- Next.js 14.2 (App Router, TypeScript)
- Prisma 7 ORM (SQLite)
- Tailwind CSS 3.4
- Nginx + systemd 部署

## 数据持久化

| 路径 | 内容 |
|------|------|
| `prisma/dev.db` | 本地开发 SQLite 数据库 |
| `/var/www/alumni-site/data/prod.db` | 线上 SQLite 数据库 |
| `public/uploads/` | 用户上传的图片（封面等） |

> 不要用本地 dev.db 覆盖线上 prod.db。线上数据变更通过后台 UI 操作。
