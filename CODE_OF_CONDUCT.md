# 行为准则

## 我们的承诺

为了营造一个开放、友好、安全的社区环境，我们承诺尊重所有参与者——无论经验水平、性别、性取向、残疾、外貌、种族、宗教或年龄。

## 我们的标准

### 鼓励的行为

- 使用友好、包容的语言
- 尊重不同观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

### 禁止的行为

- 发布骚扰、歧视、攻击性言论
- 发布他人隐私信息（真实姓名、手机号、微信号、家庭住址等）
- 发布与未成年学生相关的敏感信息
- 使用性暗示语言或图像
-  troll、侮辱、贬损性评论
- 政治或宗教攻击
- 未经许可公开他人私人信息
- 其他在专业环境中被合理认为不适当的行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并对任何不可接受行为采取适当和公平的纠正措施。

维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、commit、代码、wiki 编辑、issue 和其他贡献，并暂时或永久禁止任何贡献者做出不当、威胁、冒犯或有害行为。

## 范围

本行为准则适用于项目空间和公共空间中代表项目或其社区的场合。

## 执行

如有骚扰、虐待或其他不可接受行为，请通过 yanchuan_alumni@163.com 联系项目维护者。所有投诉将被保密审查和调查。

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.0 版。
